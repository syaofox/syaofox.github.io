MacBook Broadcom 4360 无线网卡驱动

首先，启用非免费存储库。通过将non-free 添加到您的/etc/apt/sources.list文件中。

sudo apt-get update
sudo apt-get install broadcom-sta-*
sudo modprobe wl
echo "wl" | sudo tee -a /etc/modules

不要忘记tee命令中的-a，否则您将覆盖整个文件！ 如果无法立即使用，请重新启动，应该没问题。

查看当前debian版本

cat /etc/debian_version

debian详细信息

挂载ntfs格式

安装ntfs-3g

sudo apt install ntfs-3g

命令挂载

mount -t ntfs-3g /dev/sda1 /mnt/windows

/etc/fstab

/dev/sda1 /mnt/windows ntfs-3g rw,uid=1000,gid=1000,dmask=0002,fmask=0003 0 0
