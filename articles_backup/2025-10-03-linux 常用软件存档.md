# linux 常用软件存档


### 文件管理与同步

| 工具名称 | 描述 | 主页 |
| :--- | :--- | :--- |
| **Gigolo** | 一个简单的书签界面，用于更容易访问远程文件系统，支持挂载网络硬盘（如NAS），通过GVfs挂载SMB、FTP等协议的共享文件夹。 | [访问](https://gitlab.gnome.org/GNOME/gigolo) |
| **FreeFileSync** | 免费开源的文件同步和备份软件，支持Windows、macOS和Linux，用于比较和同步本地、网络驱动器、移动设备或云存储上的文件夹。 | [访问](https://freefilesync.org/) |
| **Syncthing** | 连续文件同步程序，支持实时在多台设备间安全同步文件，使用点对点方式，无需中央服务器，适用于Windows、macOS、Linux等平台。 | [访问](https://syncthing.net/) |
| **Czkawka** | 免费开源的多功能文件清理工具，用于查找和删除重复文件、临时文件、大文件等，帮助释放磁盘空间，支持快速扫描和GUI/CLI模式。 | [访问](https://github.com/qarmin/czkawka) |
| **FSearch** | 基于GTK3的快速文件搜索工具，适用于Unix-like系统。 | [访问](https://github.com/cboxdoerfer/fsearch/wiki/Search-syntax) |
| **Clapgrep** | 开源Linux应用，用于搜索文本文件、PDF和Office文档，提供快速搜索并显示相关元数据如页码和行号。 | [访问](https://github.com/luleyleo/clapgrep) |
| **DB Browser for SQLite** | 高质量的视觉化开源工具，用于创建、搜索和编辑SQLite或SQLCipher数据库文件，支持跨平台使用。 | [访问](https://sqlitebrowser.org/) |
| **Nemo Mediainfo Tab** | 显示媒体分辨率等信息。 | [访问](https://github.com/linux-man/nemo-mediainfo-tab/releases) |

### 系统工具

| 工具名称 | 描述 | 主页 |
| :--- | :--- | :--- |
| **CopyQ** | 高级剪贴板管理器，支持监控系统剪贴板、保存历史内容、可搜索、可编辑，支持图像格式和命令行控制。 | [访问](https://hluk.github.io/CopyQ/) |
| **LocalSend** | 免费开源的跨平台应用，用于在本地网络中点对点分享文件，支持端到端加密，类似于AirDrop。 | [访问](https://localsend.org/) |
| **Resources** | 简单而强大的系统资源监视器，使用Rust和GTK 4编写，支持查看CPU、内存、磁盘使用情况和进程管理。 | [访问](https://github.com/nokyan/resources) |
| **Gear Lever** | 开源工具，用于轻松管理AppImage应用，支持安装、更新、组织AppImage文件并生成桌面入口。 | [访问](https://github.com/mijorus/gearlever) |
| **Warehouse** | Flatpak应用的图形化管理工具箱，提供简单UI来管理已安装应用、远程源和残留数据，无需命令行。 | [访问](https://github.com/flattool/warehouse) |
| **KRename** | 强大的批量文件重命名工具，支持基于表达式重命名、复制/移动文件，并处理数百个文件。 | [访问](https://apps.kde.org/krename/) |
| **FileZilla** | 免费FTP解决方案，支持FTP、FTPS和SFTP，用于远程文件传输，跨平台可用。 | [访问](https://filezilla-project.org/) |
| **slock** | 极简锁屏。 | [访问](https://tools.suckless.org/slock/) |
| **betterlockscreen** | 锁屏。 | [访问](https://github.com/betterlockscreen/betterlockscreen/) |

### 多媒体编辑

| 工具名称 | 描述 | 主页 |
| :--- | :--- | :--- |
| **Audacity** | 免费开源的多轨音频编辑和录音软件，支持Windows、macOS和Linux，用于编辑、混音和增强音频轨道。 | [访问](https://www.audacityteam.org/) |
| **Kdenlive** | 免费开源的非线性视频编辑器，基于MLT框架，支持Linux、Windows和macOS，用于专业视频制作。 | [访问](https://kdenlive.org/) |
| **Shotcut** | 免费开源的跨平台视频编辑器，支持广泛格式，无需导入即可编辑视频、音频和图像。 | [访问](https://www.shotcut.org/) |
| **Video Trimmer** | GNOME应用，用于简单无损剪辑视频，通过指定起始和结束时间戳切割视频片段，不进行重新编码以保持质量。 | [访问](https://apps.gnome.org/VideoTrimmer/) |
| **Pinta** | 免费开源的绘图和图像编辑程序，结合直观工具和强大功能，类似于简易版Paint.NET。 | [访问](https://www.pinta-project.com/) |
| **XnConvert** | 快速强大的跨平台批量图像转换器，支持自动化编辑照片集合，包括调整大小、水印和滤镜。 | [访问](https://www.xnview.com/en/xnconvert/) |
| **Upscayl** | AI图像放大工具，使用高级算法将低分辨率图像放大并增强质量，支持Linux、Windows和macOS。 | [访问](https://upscayl.org/) |
| **HandBrake** | 开源视频转码工具，支持从几乎任何格式转换为现代编解码器，适用于DVD和数字视频文件。 | [访问](https://handbrake.fr/) |
