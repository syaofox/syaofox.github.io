docker

## 以 syaofox/homer 为例

先在hub上创建名为homer的仓库

构建镜像

 docker build -t syaofox/homer:latest .

本地测试

docker run -d -p  1000:80 syaofox/homer:latest

登录hub

docker login

打标

docker tag syaofox/homer syaofox/homer:v1.0

推送到hub

docker push syaofox/homer:v1.0

私人的仓库多登录步骤

docker login

飞牛os拉取私人镜像

先ssh连接飞牛,再执行登录

docker login

如果是管理权限

sudo docker login
