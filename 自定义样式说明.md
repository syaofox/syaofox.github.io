# Jekyll Minima主题自定义样式说明

## 已实现的自定义功能

### 1. 页面宽度调整
- **默认宽度**: Minima主题默认800px
- **自定义宽度**: 已设置为1200px
- **响应式设计**: 在移动设备上自动调整为100%宽度

### 2. 字体大小调整
- **正文字体**: 从14px增加到16px
- **标题字体**: 
  - H1: 2.5em
  - H2: 2em  
  - H3: 1.5em
- **代码块**: 保持14px以确保可读性

### 3. 行高优化
- **正文行高**: 1.6-1.7，提高可读性
- **标题行高**: 保持默认，确保层次清晰

## 文件位置
自定义样式文件位于: `assets/css/style.scss`

## 如何进一步自定义

### 调整页面宽度
在 `assets/css/style.scss` 文件中修改 `.wrapper` 类的 `max-width` 值：
```scss
.wrapper {
  max-width: 1400px; // 改为你想要的宽度
}
```

### 调整字体大小
在 `assets/css/style.scss` 文件中修改 `body` 类的 `font-size` 值：
```scss
body {
  font-size: 18px; // 改为你想要的字体大小
}
```

### 调整标题大小
修改对应的标题样式：
```scss
h1 {
  font-size: 3em; // 调整一级标题大小
}
```

## 部署说明
1. 将修改后的文件提交到Git仓库
2. GitHub Pages会自动重新构建你的网站
3. 等待几分钟后刷新网站查看效果

## 注意事项
- 文件头部必须包含Jekyll front matter (`---`)
- 使用 `@import "minima";` 来继承主题的默认样式
- 自定义样式会覆盖主题的默认样式
- 建议在本地先测试样式效果再部署
